# Sistema de Ocorrências Escolar

Histórico de evolução do sistema de registro de ocorrências disciplinares
da E.E. João Paulo II.

## Versões

- [`v1-desktop`](./v1-desktop) — versão original, aplicativo desktop em
  Python (2026). Funcional e arquivada aqui como registro da primeira
  solução do problema.
- `v2` *(em construção)* — migração para backend (FastAPI) + app mobile
  (Flutter), com autenticação, histórico relacional e testes.

Este repositório é mantido como histórico contínuo do projeto — nada é
apagado entre versões.
